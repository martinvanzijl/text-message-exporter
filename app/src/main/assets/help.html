Getting Started

This app exports text messages to a file. To use it:

  1. Optionally choose a contact to filter by.
  2. Optionally choose a start date and/or end date to filter by.
  3. Click "Export Text Messages".
  4. Open the phone's File Manager.
  5. Open the top left menu (three horizontal stripes).
  6. Select the "Phone Storage" item near the bottom.
  7. Navigate to "Documents/sms-exports". There should be a file called "export.txt".

The file will have the text message details as per your chosen filters.

You can also click on the link at the top of the app to view the exported file.

Filters

By default, all text messages on the phone are exported. You can use the filters at the bottom of the page to export specific messages only.

To export messages to and from a specific contact only, click "Choose Contact", then select the contact to filter by. You can click the "Reset" button to the right to reset this filter.

To export messages from a specific date onwards, click "Choose Start Date". You can click the "Reset" button to the right to reset this filter.

To export messages only until a specific date, click "Choose End Date". You can click the "Reset" button to the right to reset this filter.

Export Options

The app will by default export a plain text (.txt) file. You can change the file type in the Settings screen. Click on "Choose exported file type", then choose the desired file type. The options are Text, XML or CSV.

Emailing the Exported File

To email the exported file as an attachment, click the button "Email Exported File". This will create a draft email message with the file attached. You simply have to add the recipient address and send it.

Previewing the Exported File

After you export text messages, you can view exported file by clicking the link above the button.

NOTE: If you try to preview the CSV file, it may not work. In my own testing, the CSV file did not display at all.

Showing the Export Progress Bar

The app can optionally show a progress bar to indicate export progress. To show this, in the Settings screen, turn on the preference "Show progress bar for export".
