Getting Started

This app exports text messages to a file. To use it:

  1. Optionally choose a contact to filter by.
  2. Optionally choose a start date and/or end date to filter by.
  3. Click "Export Text Messages".
  4. Open the phone's File Manager.
  5. Open the top left menu (three horizontal stripes).
  6. Select the "Phone Storage" item near the bottom.
  7. Navigate to "Documents/sms-exports". There should be a file called "export.txt".

The file will have the text message details as per your chosen filters.

You can also click on the link at the top of the app to view the exported file.
